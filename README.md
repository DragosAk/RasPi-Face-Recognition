 Structure:
   - collect.py: Script to capture and save face samples for new users.
  
   - train.py: Trains the recognizer model using the collected face samples.
  
   - run.py: The main script to run the face recognition and attendance tracking.
  
   - clear.py: Utility script to clear existing data or reset the logs.

 Requirements:
   - Webcam
   - OpenCV
   - Numpy
   - Python 3.x

 Setup: 
   - git clone https://github.com/DragosAk/RasPi-Face-Recognition
   - pip install opencv-contrib-python numpy
